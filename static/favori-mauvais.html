Mauvais favori InfoBox

Votre favori pointe vers cette page (/assistant ou l’URL du site) au lieu d’un lien javascript:…. C’est pour cela que BoxRec vous renvoie ici.

Comment corriger

  1. Supprimez le favori « InfoBox » actuel dans Chrome / Edge.
  2. Ouvrez l’Assistant InfoBox.
  3. Glissez le bouton bleu vers la barre de favoris — ne mettez pas cette page en favori avec l’étoile.
  4. Sur boxrec.com (liste managers, etc.), cliquez le nouveau favori : vous devez rester sur BoxRec.
